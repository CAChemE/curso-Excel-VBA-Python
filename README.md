  * Importación y exportación de datos

  * Funciones VBA

* Sesión 4:
  * Introducción a Jupyter Notebooks
  * Introducción a pandas

* Sesión 5:
  * Ejecución de código Python desde Excel (xlwings)
  * Sustituyendo VBA por Python (pandas)
